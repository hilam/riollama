# Riollama

Chatbot com Llama3 via Ollama, criado usando [Rio](http://rio.dev/).

Projeto baseado no templete `AI Chatbot`.

## AI Chatbot

Um chat simples com uma IA Llama3.

**Não requer nenhuma senha ou chave de API.**

## Requisitos

-   Instalar Ollama
-   Baixar modelo
-   Instalar dependências
